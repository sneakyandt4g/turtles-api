# turtles-api
An RESTful turtle web API that uses JSON.

This API is made using Flask!

## Get a fact using turtles-api.
(GET) /fact

Gets a fact.

[Return this right now](https://sneakys.pythonanywhere.com/api/turtles/v1/fact)

## Return the status of the API.
(GET) /

Returns the status.
